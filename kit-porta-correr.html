Kit Porta de Correr

O Kit Porta de Correr é a solução ideal para ambientes que necessitam otimização de espaço, combinando funcionalidade, design moderno e praticidade. Este sistema de abertura deslizante não requer espaço para abertura da porta, sendo perfeito para ambientes compactos ou layouts que exigem melhor aproveitamento de área.

Composto por trilho, folha de porta, roldanas de alta qualidade e todos os acessórios necessários, o kit porta de correr oferece instalação simplificada e movimentação suave e silenciosa. É uma opção versátil que une economia de espaço com estética contemporânea.

Tipos de Sistema

Oferecemos diferentes sistemas para atender cada necessidade:

  • Externo (Sobrepor): Trilho aparente na parede externa
  • Interno (Embutir): Porta desliza dentro da parede
  • Duas Folhas: Abertura central mais ampla
  • Uma Folha: Solução compacta e econômica
  • Com Trilho Inferior: Maior estabilidade
  • Sem Trilho Inferior: Limpeza facilitada

Dicas de Instalação

  • Verifique a resistência da parede para suportar o trilho
  • Instale o trilho perfeitamente nivelado
  • Use buchas adequadas ao tipo de parede
  • Regule as roldanas para altura uniforme
  • Teste o deslizamento antes da fixação final
  • Instale batedor para proteção da parede
  • Para sistema embutido, prepare o vão durante a obra

Manutenção

  • Limpe o trilho regularmente para remover sujeira
  • Lubrifique as roldanas a cada 6 meses
  • Verifique o aperto dos parafusos periodicamente
  • Ajuste a altura conforme necessário
  • Evite impactos e movimentos bruscos

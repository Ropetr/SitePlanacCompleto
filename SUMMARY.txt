  1. download-images.py       - Baixa 42 imagens automaticamente
  2. optimize-images.py       - Converte para WebP e otimiza
  3. upload-to-r2.py          - Upload para Cloudflare R2
  4. update-image-links.py    - Atualiza links nos HTMLs

📚 Documentação (4):
  1. IMAGES-QUICKSTART.md     - Guia rápido (COMECE AQUI!)
  2. cloudflare-r2-setup.md   - Setup Cloudflare R2 detalhado
  3. assets/images/README.md  - Doc do sistema de imagens
  4. README.md                - README principal atualizado

📝 Configuração (1):
  1. requirements.txt         - Dependências Python

================================================================================
🚀 COMO USAR (3 MINUTOS):
================================================================================

1️⃣ Instalar dependências:
   pip install -r requirements.txt

2️⃣ Baixar e otimizar imagens:
   python download-images.py
   python optimize-images.py

3️⃣ Escolher hospedagem:

   OPÇÃO A - Cloudflare R2 (RECOMENDADO):
   • CDN global grátis
   • PageSpeed 90-100
   • Praticamente $0/mês
   
   Passos:
   - npm install -g wrangler
   - wrangler login
   - python upload-to-r2.py
   - python update-image-links.py (escolha opção 1)

   OPÇÃO B - Local (MAIS SIMPLES):
   • Zero configuração
   • Deploy com as imagens
   
   Passos:
   - python update-image-links.py (escolha opção 2)

4️⃣ Testar:
   python -m http.server 8000
   Abra: http://localhost:8000
